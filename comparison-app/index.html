Framework Comparison

Vue.js Characteristics

  • Template Syntax: HTML-like templates with directives
  • Reactivity: Built-in reactive data system
  • Component Structure: Single File Components (.vue)
  • State Management: data() function returns reactive state
  • Event Handling: @click directive syntax
  • Conditional Rendering: v-if, v-else directives
  • List Rendering: v-for directive
// Vue component structure
<template>
  <div @click="handleClick">
    <img v-for="dog in dogs" :src="dog.url" />
  </div>
</template>

<script>
export default {
  data() {
    return {
      dogs: [],
      currentIndex: 0
    }
  },
  methods: {
    handleClick() { /* ... */ }
  }
}
</script>

React.js Characteristics

  • JSX Syntax: JavaScript XML for component templates
  • Hooks: useState, useEffect for state and lifecycle
  • Component Structure: JavaScript/JSX files
  • State Management: useState hook for component state
  • Event Handling: onClick prop syntax
  • Conditional Rendering: JavaScript ternary operators
  • List Rendering: Array.map() method
